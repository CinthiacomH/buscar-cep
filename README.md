# buscar-cep
Buscador de CEP com VBA

Para que o código funcione, é necessário criar um arquivo Excel com as colunas:
"CEP" na posição A:3;
"Logradouro/Nome" na posição B:3;
"Bairro/Distrito" na posição C:3;
"Localidade/UF" na posição D:3.

Em seguida abra o VBA no excel com as teclas Alt + F11 ou Alt + Fn + F11 e cole o código.
